# Sistema de Cadastro de Usuários

Projeto desenvolvido com HTML, CSS e JavaScript puro para realizar o cadastro de usuários através de uma API REST.

##  Funcionalidades

* Cadastro de novos usuários
* Validação de campos obrigatórios
* Envio de dados para API utilizando Fetch API
* Feedback visual de sucesso e erro
* Limpeza automática do formulário após cadastro

##  Tecnologias Utiladas

* HTML5
* CSS3
* JavaScript (ES6+)
* Fetch API
* MockAPI

##  Interface

A aplicação possui uma interface moderna, responsiva e intuitiva, utilizando:

* Design Dark Mode
* Tipografia com Google Fonts
* Animações suaves em botões e links
* Layout centralizado com Flexbox

## ⚙️ Como Executar

1. Clone este repositório:

```bash
git clone <url-do-repositorio>
```

2. Acesse a pasta do projeto:

```bash
cd Cadastro-Usuarios
```

3. Abra o arquivo `index.html` no navegador.

## 🔗 API Utilizada

A aplicação utiliza a plataforma MockAPI para simular um backend REST.
